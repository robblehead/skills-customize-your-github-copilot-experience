# 📘 Assignment: Python Fundamentals

## 🎯 Objective

Practice beginner Python skills by working with variables, user input, conditionals, and loops. By the end of this assignment, students will write clear, interactive programs that solve simple real-world tasks.

## 📝 Tasks

### 🛠️ Build a Student Introduction Program

#### Description
Create a program that asks for a student's name, grade level, and favorite subject, then prints a formatted summary sentence.

#### Requirements
Completed program should:

- Ask for `name`, `grade level`, and `favorite subject` using `input()`.
- Store each answer in a separate variable.
- Print one formatted sentence using all three values.
- Keep output readable and student-friendly.


### 🛠️ Make a Grade Checker

#### Description
Write logic that reads a numeric score and prints whether the student passed or needs more practice.

#### Requirements
Completed program should:

- Ask the user for a score from 0 to 100.
- Convert the input to an integer.
- Print `Pass` when score is 70 or higher.
- Print `Needs more practice` when score is below 70.
- Handle edge cases such as exactly 70 correctly.


### 🛠️ Create a Simple Progress Tracker

#### Description
Use a loop to count completed practice problems and show progress updates.

#### Requirements
Completed program should:

- Ask how many practice problems the student completed.
- Use a `for` loop to print progress messages from 1 up to that number.
- Print a final congratulations message after the loop ends.
- Ensure the loop works for small numbers such as 1, 2, and 3.
